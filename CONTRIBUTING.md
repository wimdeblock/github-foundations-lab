# Contributing Guide

Bedankt dat je wilt bijdragen aan deze repository.

## Werkwijze

1. Maak een nieuwe branch.
2. Voer je wijziging door.
3. Maak een pull request.
4. Vraag review aan.

## Branch naming

Gebruik de volgende structuur:

```text
feature/<omschrijving>
bugfix/<omschrijving>
docs/<omschrijving>
```

## Commit messages

Gebruik duidelijke commit messages, bijvoorbeeld:
```text
Add installation instructions
Fix typo in README
Update project documentation
``
